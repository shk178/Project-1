### 공통
- 푸드 트럭 당 키오스크 1대
- 손님 모드, 직원 모드 화면 다름
- :q 입력 시 이전 화면으로 간다.
### 손님 모드
- 메인 화면
```
🏠
👉 [주문 시작하기] 주문을 시작하려면 엔터 키를 누르세요.
```
- 메뉴 선택 화면 (메인 화면에서 엔터 키)

### 직원 모드
- 로그인 및 메인 화면
```
👉 관리자 화면으로 이동하려면 로그인 하세요.
🔒[관리자 인증] 관리자 로그인 아이디를 입력해 주세요.
admin
🗝[관리자 인증] 패스워드를 입력해 주세요.
steak1111
‼️ 비밀번호가 일치하지 않습니다.
👉 관리자 화면으로 이동하려면 로그인 하세요.
🔒[관리자 인증] 관리자 로그인 아이디를 입력해 주세요.
admin
🗝[관리자 인증] 패스워드를 입력해 주세요.
admin1234
🔓 인증되었습니다!
🏠
==== 관리자 메뉴 ====
1. 주문 관리
2. 매출 관리
👉 관리자 메뉴의 번호를 입력하세요. ex) 1
1
```
